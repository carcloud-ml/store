### BoxJS 无法连接问题总结：

- 检查代理软件是否启用
- 检查设置里阻止跨域跟踪是否关闭
- 检查代理软件中是否将 GitHub 加速关闭
- 检查 MitM 中是否存在当前 boxjs 域名（该网站为 https 访问，boxjs 的关联也必须为 https）
- 更多问题请到[电报群](https://t.me/Scriptable_JS)
